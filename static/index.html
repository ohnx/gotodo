    Token Management

    Tokens are created in order to access the gotodo API. When you log in, you are creating a master-access token, which is then used for other manipulations of the API.

    If you would like, you can also create more tokens so that other applications can pretend like they are you and access the gotodo API to perform operations like creating, modifying, and deleting todo items.

    This panel allows management of tokens. To create a new token, press the "create token" button. A new token, with the specified permissions, will be created. To invalidate an existing token, paste in a token into the token text field and press the "invalidate token" button.
